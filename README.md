## ⚡ TÍNH NĂNG CHÍNH

### ✅ Website tĩnh siêu nhanh
- HTML/CSS thuần, không database
- Tốc độ load < 2 giây
- SEO score 95-100
- Mobile-friendly

### ✅ CMS quản lý không cần code (Decap CMS)
- Login tại `/admin` bằng email
- Thêm/sửa: sản phẩm, bài viết, FAQ, slider, hotline...
- Upload ảnh kéo thả
- Lịch sử backup tự động trên GitHub

### ✅ Form liên hệ (Netlify Forms)
- Khách điền form → bạn nhận email ngay
- Anti-spam tự động
- 100 submissions/tháng miễn phí

### ✅ Đặt lịch online (Cal.com)
- Khách tự book lịch khảo sát/tư vấn
- Tự động sync với Google Calendar
- SMS/Email nhắc nhở

### ✅ SEO chuẩn
- 8+ Schema.org types
- FAQ Rich Snippets
- Sitemap auto-update khi đăng bài mới
- HSTS + Security headers
